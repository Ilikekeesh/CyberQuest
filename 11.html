Level 1.1: Common Threats


Goals:
  • Learning about common issues and cybersecurity threats
  • Building a mental dictionary of important security terms

In the world of cybersecurity, we deal with a lot of different types of threat actors. In this level, we will learn about the most common types and how they act.

Software Attacks
  • Denial of service (or distributed denial of service)
  • Computer worms, viruses, and other malware
Software Errors
  • Software bugs and human error
  • Applications going offline
  • Cross-site scripts or illegal file server sharing
Sabotage
  • Authorized users penetrating and compromising data
  • Defacement of a organization's website
Human Error
  • Inadvertant data entry error
  • Firewall misconfigurations
Theft
  • Laptops or other data stolen from unlocked rooms
Hardware Failure
  • Hard drive crashes
Utility Interruption
  • Electrical power outage
  • Water damage
Natural Disasters
  • Hurricanes, tornadoes
  • Thunderstorms, flooding
  • Fires, earthquakes
